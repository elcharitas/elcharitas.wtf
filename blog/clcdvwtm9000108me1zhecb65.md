# Chakra UI Svelte v0: What to look forward to this New Year

I'll start by saying a **Happy New Year** to you, my most esteemed reader 😎, the previous year was a blast thanks to your continual reading of my write ups. If this is your first time here or you're not yet subscribed to this blog, then you really should 😩.  
It's what keeps me going 😌.

Now, down to business... Chakra UI-Svelte is an open-source component library for building user interfaces with the Svelte framework. It is based on the popular [Chakra UI library](https://chakra-ui.com), which is built with [React](https://reactjs.org). I've been working on new features and improvements to the library, and several exciting updates are coming soon.

* Improved performance and optimizations: The Chakra UI-Svelte team is constantly working to improve the performance of the library, and the next release will include several optimizations to make it even more efficient and effective. These optimizations will help developers build faster and more responsive applications with Chakra UI-Svelte.
    
* New documentation site powered by [Histoire](http://histoire.dev): The new documentation site will make it easier for developers to find and learn about the various components and features offered by Chakra UI-Svelte. It will include detailed documentation, examples, and code snippets to help developers get up and running with the library quickly and easily.

* New components: I am working on adding new and useful components to the library. In the next release, users can expect to see several new components that will help them build even more powerful and feature-rich applications. The goal is to have 40 ready reusable components by the end of the first quarter.
    
* Better accessibility support 😳: Accessibility is a top priority for the Chakra UI-Svelte, and I am also working to improve the library's support for users with disabilities. In the next release, users can expect to see even better support for screen readers and other accessibility tools.

* Other improvements and enhancements 👌: In addition to the major updates listed above, users can expect several other improvements and enhancements to the library. These include new features and functionality, as well as bug fixes and other enhancements to make the library even better.
    

Overall, these new features and improvements will make Chakra UI-Svelte an even more powerful and useful tool for building user interfaces with the Svelte framework. Whether you're a seasoned developer or new to Svelte, you'll find a lot to love in the upcoming updates to Chakra UI-Svelte.
